# CSS
CSS는 Cascading Style Sheets의 약자로, 웹 페이지의 스타일을 정의하는 언어입니다.
HTML과 함께 사용되며, HTML 요소들의 스타일으 정의하는데 사용됩니다.

## CSS3에서 추가된 새로운 기능

 - 그리드 레이아웃
 - 플렉스 박스
 - 다단 Drop Cap
   - 텍스트의 첫 글자를 크게 만들 어 다단의 시작 부분을 장식할 수 있습니다.
 - 그림자
 - Transform
 - 새로운 셀렉터
   - :not(), :nth-child(), :first-child()
 - 애니메이션과 트랜지션 같은 기능
